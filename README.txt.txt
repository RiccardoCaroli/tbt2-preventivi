TBT2 SERRAMENTI - GUIDA RAPIDA PER LA WEBAPP PREVENTIVI
========================================================

Autore: TBT2 Serramenti s.r.l.
Versione: 1.0 – 2025
Sito web: www.tbt2.com

--------------------------------------------------------
1️⃣  COME APRIRE LA WEBAPP
--------------------------------------------------------
• Apri il file "index.html" in Safari, Chrome o Edge.
• Su iPad puoi tenere premuto sul file → “Condividi” → “Apri in Safari”.
• Ti apparirà la schermata principale del generatore preventivi TBT2.

--------------------------------------------------------
2️⃣  COME AGGIUNGERE L'APP ALLA HOME DELL'iPAD
--------------------------------------------------------
• In Safari, tocca il pulsante di condivisione (quadrato con freccia verso l’alto).
• Scorri in basso e seleziona “Aggiungi alla schermata Home”.
• L’app apparirà come icona blu con il logo TBT2.

--------------------------------------------------------
3️⃣  COME USARE IL GENERATORE
--------------------------------------------------------
• Inserisci i dati cliente nei campi in alto.
• Nella tabella aggiungi o rimuovi righe con il pulsante “+ Aggiungi Riga”.
• Compila larghezza, altezza e prezzo €/m².
• Premi “Calcola Totale” per aggiornare i valori.
• Premi “Genera PDF” per salvare o stampare il preventivo.

--------------------------------------------------------
4️⃣  COME PERSONALIZZARE
--------------------------------------------------------
• Puoi modificare il logo sostituendo il file "Logo-TBT2.png".
• Puoi aggiornare i colori cambiando i valori HEX nel codice HTML (#003366).
• La versione e l’anno si trovano in fondo al file (piè di pagina).

--------------------------------------------------------
5️⃣  COME PUBBLICARE ONLINE (GITHUB PAGES)
--------------------------------------------------------
1. Crea un account su github.com.
2. Crea un nuovo repository (es. "tbt2-preventivi").
3. Carica il file "index.html" e le eventuali immagini (Logo, favicon, ecc.).
4. Vai su "Settings" → "Pages" → “Deploy from branch”.
5. Dopo qualche secondo, il tuo sito sarà disponibile all’indirizzo:
   https://tuonomeutente.github.io/tbt2-preventivi/
